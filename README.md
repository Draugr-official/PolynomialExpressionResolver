# Polynomial expression resolver.

This application takes an x-axis array and a y-axis array, then solves the original polynomial expression used for the given graph.

Example;
```cs
double[] xAxis = { 0, 1, 2, 3, 4, 5 };
double[] yAxis = { 30, 32, 34, 36, 38, 40 };
```

Would resolve as the following;

``2*x + 30``
